# 🏛️ Civic Issue Portal

A full-stack web application that helps citizens report civic problems such as potholes, drainage issues, garbage overflow, water leakage, and streetlight failures directly to authorities.

---

## 🌐 Project Overview

In many areas, people face daily civic problems but there is no proper digital platform to report them efficiently.

The **Civic Issue Portal** provides a platform where citizens can:

- Report civic problems with images
- Track complaint status
- View updates from authorities
- Help improve communication between citizens and administration

This system also includes geographical boundary validation to accept reports only from within Andhra Pradesh.

# ✨ Features

- 🔐 User Authentication & Authorization
- 🛡️ Role-Based Access (User/Admin)
- 📍 Andhra Pradesh Geographical Boundary Validation
- 🖼️ Image Upload using AWS S3
- 📌 Report Civic Problems
- 📊 Complaint Status Tracking
- 👨‍💼 Admin Dashboard for Managing Complaints
- 📱 Responsive UI
- ⚡ REST API Integration
- 🗄️ MySQL Database Integration
